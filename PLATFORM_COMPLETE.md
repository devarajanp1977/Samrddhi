## 🛠️ Development Workflow

```bash
# Development mode
docker-compose up --build -d

# View service logs
docker-compose logs -f [service-name]

# Restart specific service
docker-compose restart [service-name]

# Stop all services
docker-compose down

# Clean rebuild
docker-compose down -v && docker-compose up --build -d
```
